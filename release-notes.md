## SkyWeather v0.2.0

### 中文版
第二个预览版本发布！

**新增功能：**
- 新增 Android 桌面小组件支持
  - 天气概览小组件：显示当前天气和未来 5 小时温度趋势
  - 日出日落小组件：显示日出日落时间
  - 空气质量小组件：显示空气质量指数
- 小组件配置页面，支持自定义显示内容
- 小组件定时更新与后台数据同步
- 降水概率切换服务

**优化与修复：**
- 优化应用整体性能
- 提升天气数据加载速度
- 修复已知问题

**安装方式：**
下载 `SkyWeather-v0.2.0.tar.gz`，解压后安装其中的 `app-debug.apk`。

### English
Second preview release!

**New Features:**
- Added Android home screen widget support
  - Weather Overview Widget: displays current weather and 5-hour temperature trend
  - Sunrise & Sunset Widget: displays sunrise and sunset times
  - Air Quality Widget: displays air quality index
- Widget configuration page with customizable content
- Widget auto-update and background data sync
- Precipitation probability switch service

**Improvements & Fixes:**
- Optimized overall app performance
- Improved weather data loading speed
- Fixed known issues

**Installation:**
Download `SkyWeather-v0.2.0.tar.gz`, extract it, and install `app-debug.apk`.
