## SkyWeather v0.2.2

### 中文版
- 新增小尺寸天气概览小组件（1×1），支持快速查看当前温度与天气状况
- 重新设计天气概览小组件布局，支持更大尺寸展示，信息更丰富
- 小组件新增展开/收起动画效果，提升交互体验
- 新增深色模式小组件适配，随系统自动切换
- 优化小组件温度趋势图表渲染效果
- 修复降水概率小组件数据同步问题

### English
- Added small weather overview widget (1×1) for quick temperature and weather glance
- Redesigned weather overview widget layout with larger size support for richer information
- Added expand/collapse animations to widgets for better interaction
- Added dark mode support for widgets, following system settings
- Optimized temperature trend chart rendering in widgets
- Fixed data sync issue in precipitation probability widget
